# Find where MCP OAuth breaks — fast.

AuthProbe is a CLI that pinpoints where MCP OAuth breaks and provides deterministic findings and verification commands.

[Get Started](/docs/getting-started/) · [Install AuthProbe](/docs/install/) · [View on GitHub](https://github.com/authprobe/authprobe)

## Why teams use AuthProbe

- Pinpoint exactly where OAuth flow failures happen.
- Get deterministic findings you can share with your team.
- Use verification commands to confirm remediations quickly.

## Start in minutes

1. Install the CLI.
2. Run a targeted scan against your MCP integration.
3. Follow the output and verification commands to validate fixes.

Visit the [Quickstart](/docs/quickstart/) to run your first scan.
